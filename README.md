# MORS
### MORS (modern overly resourceful system) is a simple morse code translator.

## Features

- Text 2 Morse Code translation
- Morse Code 2 Text translation
- Morse Code audio generator

## Usage

### Open mors.exe and you are provided with a command line interface, commands are as follows: `txt`(T2M), `morse`(M2T), `audio`(M2WAV) and `exit`.
